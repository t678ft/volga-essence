# VOLGA ESSENCE

Одностраничный сайт-магазин нишевой парфюмерии. Люкс-модерн с элементами арт-деко, мраморный фон, шампанское золото, светлая и тёмная темы.

## Возможности

- Hero с кастомной типографикой (Italiana) и золотым декором
- Каталог из 8 ароматов, вдохновлённых городами Поволжья, с фильтрами по семействам
- Модальное окно заказа: выбор объёма → контакты → карта СДЭК (Leaflet) → оплата (СБП / Картой / Mir Pay) → подтверждение
- Фиктивная генерация QR, таймер 5 минут, золотое конфетти
- Переключатель светлой/тёмной темы (белый и чёрный мрамор)
- Кастомный золотой курсор, частицы золотой пыли, пасхалка на долгом наведении на логотип

## Запуск

Просто откройте `index.html` в браузере. Внешних зависимостей две: Leaflet (карта) и Google Fonts — подгружаются по CDN.

## Стек

Чистый HTML + CSS + JavaScript, без сборки.
